# 投研文章视觉AI模块

## 封面图（3 组创意）

### 创意类型

| 创意 | 策略 | 适合文章类型 |
|------|------|-------------|
| A：K线走势型 | 用 K 线视觉隐喻表达文章核心判断 | 技术面、价值锚定、事件驱动 |
| B：产业链地图型 | 结构化信息图展示产业链位置 | 产业链洞察、对比分析 |
| C：情绪氛围型 | 场景渲染表达投资情绪 | 实盘复盘、财报解读 |

### 实体锚定规则

从文章中提取 3-5 个具体实体：
- 股票代码/名称
- 行业/概念关键词
- 关键数据（PE/增速/市值）
- 情绪/场景（恐慌/冲高回落/怀疑）

**硬规则**：每条提示词至少包含 2 个文章实体。禁止纯抽象描述。

### 提示词示例

```
### 封面创意 A: K线关键位置
- 视觉描述：日K线图在年线位置放量突破，成交量柱状图显著放大，
  红色K线从均线束中向上穿刺，背景是柔和的数据网格。
- 色调：{行业色 + 强调色}
- 构图：16:9，K线图占据画面左侧 2/3，右侧 1/3 留白放标题
- AI 绘图提示词：
  "Stock K-line chart breaking above the 200-day moving average,
  candlestick chart with red candles, volume bars showing significant increase,
  minimalist financial visual style, dark background with subtle grid,
  16:9 aspect ratio, clean space on right third for text overlay,
  no text no letters no words"
- 适配工具：Midjourney/即梦
```

## 内文配图

### 图片类型

| 类型 | 内容模板 |
|------|---------|
| **走势图** | `{股票名称} {时间段} {关键均线/信号} 走势，{成交量/资金流}对比，清新风投资图表，16:9，无文字` |
| **对比图** | `左右对比，左侧{公司A} {数据维度和值}，右侧{公司B} {数据维度和值}，简洁对比布局，16:9` |
| **信息图** | `结构化信息图展示{行业名称}产业链，从{上游}到{下游}，{公司名称}位于{位置}，16:9，无文字` |
| **场景图** | `投资情绪/决策场景，{具体场景描述}，{情绪关键词}氛围，16:9` |
| **流程图** | `决策流程图：{条件1}→{条件2}→{结论}，简洁箭头连接，16:9` |

### 通用要求

- 尺寸统一 16:9
- 所有提示词末尾附加视觉锚点风格关键词
- 每张图提供免费图库备选关键词
