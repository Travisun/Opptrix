# AStockWriter 合规红线

## 中国法律法规要求

根据《证券法》、《关于规范证券投资咨询机构及执业人员执业行为的暂行规定》以及
微信公众平台运营规范，投研类公众号文章必须严格遵守以下三条红线：

---

## 三条红线（违者封号）

### 红线一：不得推荐个股

**禁止的行为**：
- 明确说"买"、"卖"、"持仓"、"建仓"某只股票
- 暗示"XX 股票值得关注"、"XX 可以入手"
- 通过"XX 我已经买了"变相推荐（展示个人持仓隐含推荐）
- 通过问句暗示："XX 这个位置是不是可以买了？"

**可以做的**：
- 分析商业模式、财务数据、行业格局
- 展示数据事实："PE 处于历史 XX% 分位"、"毛利率从 XX% 变化到 XX%"
- 分享思考框架："如果你是 XX 类型的投资者，可以关注 XX、YY、ZZ 几个维度"

### 红线二：不得预测个股未来走势

**禁止的行为**：
- 预测股价："看到 XX 元"、"目标价 XX"
- 预测涨跌："XX 接下来要涨/跌"、"XX 有翻倍潜力"
- 预测时间："下个季度 XX 一定会 XX"、"三个月的窗口期"
- 隐含预测："从历史规律看，XX 接下来应该 XX"

**可以做的**：
- 分析过去的数据和趋势（"过去 3 年营收复合增长率 XX%"）
- 讨论行业景气度变化（非个股）
- 指出需要关注的关键节点（"可以关注 XX 指标在下个季度的变化"）

### 红线三：不得引导他人投资具体个股

**禁止的行为**：
- "这个位置可以布局了"、"建议关注 XX"
- "如果是我，我会……"（当上下文指向投资决策时）
- "大家注意，XX 的机会来了"
- 用"操作建议"、"买卖点"等投资建议语言

**可以做的**：
- "我的个人操作记录是……（这是个人行为，不是建议）"
- "我做对了什么/做错了什么"（复盘视角）
- 财务数据分析和对比

---

## 合规写作原则

### 数据就是数据，不是信号

> 展示 PE 处于历史低位，不等于在说"可以买了"。
> 展示 MACD 金叉，不等于在说"要涨了"。
> 差一个字，就是合规和违规的区别。

**正确表述**："XX 的 PE 目前处于近 5 年的第 XX 百分位。"
**违规表述**："XX 现在 PE 这么低，可以关注了。"

**正确表述**："MACD 在 XX 位置出现金叉信号，这是技术指标层面的一个观察。"
**违规表述**："MACD 金叉了，XX 要涨。"

### 用"分析框架"替代"投资建议"

| 违规写法 | 合规改写 |
|---------|---------|
| "XX 值得买入" | "从 ROE 和自由现金流的角度看，XX 的财务质量具备讨论价值" |
| "XX 的目标价看到 XX" | "按 DCF 模型估算，XX 在不同增长率假设下的估值区间为 XX-YY" |
| "建议关注 XX" | "可以关注 XX 行业中这些关键指标的变化" |
| "XX 要涨了/要跌了" | "市场对 XX 的预期主要分歧在 XX 和 YY 两点" |
| "XX 是机会" | "XX 行业正在经历 XX 变化，这对行业内各公司的影响各有不同" |

### 持仓披露的合规处理

如果文章涉及个人持仓（如实盘复盘）：

**必须做的**：
1. 明确说明"这是我的个人操作记录，不构成任何投资建议"
2. 以复盘/反思视角书写（"我做错了什么"），不暗示他人跟随
3. 不展示当前持仓（尤其是浮盈状态），或展示时配足风险警告

**严禁做的**：
1. 用盈利截图暗示投资水平，诱导他人跟随
2. 在复盘末尾说"操作建议"或"下次可以这样做"
3. 展示买卖点作为"信号"

### 标题合规

**合规标题**（分析视角）：
- "XX 的 PE 跌到历史低位——数据解读"
- "XX 的财报拆解：营收增长的含金量有多高？"
- "从 XX 的产业链变化看行业逻辑"

**违规标题**（推荐/预测视角）：
- "XX 跌到底了，可以抄底了吗？"（暗示"可以"）
- "XX 目标价 XX，还有 XX% 空间"（预测）
- "XX 是下一个 10 倍股"（引导投资）

### 合规风险声明

每篇文章结尾必须包含以下风险声明（不修改措辞）：

```
---
**免责声明**：本文仅为个人投研思考记录，不构成任何投资建议。
文中所有分析均基于公开可查的数据和个人理解，不保证准确性。
股市有风险，投资需谨慎。任何投资决策请独立判断。
```

**严禁**在正文中穿插"不构成投资建议"作为免责背书——必须有独立的声明段落。

---

## 合规检查表

发布前逐项检查：

- [ ] 全文没有出现"买"、"卖"、"建仓"、"加仓"、"减仓"、"关注(具体个股)"
- [ ] 全文没有出现"目标价"、"看到 XX 元"、"XX 倍空间"等价格预测
- [ ] 全文没有出现"可以入手"、"值得布局"、"机会来了"等引导性表达
- [ ] 全文没有出现"我推荐"、"建议关注"、"建议买入"等推荐语言
- [ ] 标题没有暗示投资决策（"抄底"、"机会"、"翻倍"等）
- [ ] 如果涉及个人持仓，有明确的"个人记录，非建议"声明
- [ ] 有独立的、完整格式的免责声明段落
- [ ] 分析框架以"行业/业务/财务"为核心，不以"涨/跌/买/卖"为核心
