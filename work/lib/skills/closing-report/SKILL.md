# 收盘报告 (Closing Report)

A股收盘后（15:30 后）生成，回顾当日市场全貌、复盘交易、规划次日。

## 快速使用

```python
from a_stock_layer import AshareEngine
from skills.closing_report.report_closing import ClosingReport

engine = AshareEngine()
reporter = ClosingReport(engine)
print(reporter.generate())
```

## 数据采集维度

| # | 维度 | API 调用 | 说明 |
|---|------|----------|------|
| 1 | 主要指数 | `index_realtime("000001/399001/399006/000688/000016/000300/000905/000852")` | 全市场八大指数 |
| 2 | 市场广度 | `market_breadth()` | 涨跌家数、涨跌占比、新高新低 |
| 3 | 涨停跌停 | `limit_updown()` | 涨停/跌停/炸板/连板/概念分布 |
| 4 | 北向资金 | `market_money_flow("north")` | 当日净流入 + 连续方向 |
| 5 | 行业资金流 | `sector_money_flow("industry")` | 各行业主力净流入+领涨股 |
| 6 | 龙虎榜 | `dragon_tiger()` | 机构买卖、龙虎榜净额 |
| 7 | 可转债 | `convertible_bonds()` | 折价套利、低溢价活跃 |
| 8 | 持仓盈亏 | `portfolio.summary()` / `portfolio.holdings()` | 整体+逐只盈亏 |
| 9 | 持仓公告 | `news(code)` | 持仓股当日公告 |

## 分析逻辑

### 风格轮动分析
通过比较不同指数涨跌幅判断当日市场风格：
- 沪深300 vs 中证1000: 大票 vs 小票
- 创业板 vs 深证成指: 成长 vs 价值

### 涨停深度分析
| 指标 | 含义 |
|------|------|
| 涨停家数 | 市场情绪温度计 |
| 最高连板 | 市场最高标，短线情绪锚 |
| 连板股统计 | 热点持续性判断 |
| 概念分布 | 当日主线题材 |
| 炸板家数 | 情绪分歧程度 |

### 龙虎榜机构识别
通过席位名称包含"机构专用"来识别机构操作：
- 机构净买入Top3: 机构看好的标的
- 机构净卖出Top3: 机构撤退的标的
- 龙虎榜总净额Top5: 市场关注焦点

### 市场状态综合评分
六因子打分系统：

| 因子 | +2分 | +1分 | -1分 | -2分 |
|------|------|------|------|------|
| 涨跌比 | >3:1 | >2:1 | <0.8:1 | <0.5:1 |
| 涨停家数 | >80 | >40 | <10 | — |
| 跌停家数 | — | — | >20 | >50 |
| 成交额 | >1.5万亿 | — | <7000亿 | — |
| 北向流入 | >50亿 | — | — | <-50亿 |

- **≥5分**: 强势 — 积极操作
- **2~4分**: 偏强 — 结构性机会
- **-1~1分**: 震荡 — 控制仓位
- **-3~-2分**: 偏弱 — 防御为主
- **≤-4分**: 弱势 — 谨慎观望

### 量价分析
- 成交额 > 1.5万亿: 放量活跃
- 成交额 8000亿~1.2万亿: 存量博弈
- 成交额 < 8000亿: 缩量观望

### 可转债机会识别
- 折价转股套利: 溢价率 < 0%
- 低溢价活跃: 溢价率 < 10% 且价格 < 120

## 输出结构

```
# A股收盘报告 | YYYY-MM-DD

## 大盘表现
[八大指数表格]
[涨停跌停分析 + 连板 + 概念分布]
[风格轮动判断 + 市场广度 + 量能]

## 市场判定
[强势/偏强/震荡/偏弱/弱势 + 操作建议]

## 资金流向
[北向资金]
[行业资金流入Top3 + 流出Top3 + 领涨股]

## 龙虎榜
[机构净买入/卖出 + 龙虎榜净额Top3]

## 持仓复盘
[逐只持仓表格 + 今日涨跌 + 累计盈亏 + 异常提醒]
[整体盈亏汇总]

## 持仓相关公告
[持仓股当日重要公告摘要]

## 可转债观察
[折价套利/低溢价活跃标的]

---
## 明日关注
[技术面/资金面/风格/情绪/板块 N 个关注点]
```

## 什么时候触发

- 用户问："收盘报告" "今天怎么样" "复盘"
- 交易日 15:30 之后
- 非交易日可回答："今天不开盘，看一下最近交易日的数据吧？"

## 注意事项

- 所有数据采集使用 `concurrent.futures.ThreadPoolExecutor` 并行拉取
- 无持仓时正常生成除持仓外的完整报告
- 个别维度API失败不影响整体输出
- 北向资金数据需要东方财富API可用时才有
