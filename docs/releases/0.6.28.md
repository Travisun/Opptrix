## 新功能

- 首次启动与版本升级后的**启程引导**：连接大模型、确认行情来源、阅读协议后即可开始投研
- 启动时展示**随机投资忠告**，轻量进入工作状态
- 多模态设置支持**按需下载**离线翻译模型，默认不占用额外空间
- 市场动态页看板：涨幅、跌幅、美股龙头与关注列表同屏浏览
- GitHub 发布页自动附带**本版更新说明**（新功能与修复清单）

## 修复

- 市场动态页在 Windows / Linux 上隐藏多余滚动条，界面更干净
- 聊天流式输出时侧栏与右栏不再抖动
- 侧栏设置按钮可正确打开「通用」设置
- 桌面端启动时本地端口冲突更易自动化解
- 聊天启动与列表展示更稳定，减少偶发白屏或重复 key 警告
- 本地行情与因子库升级时自动迁移，旧数据可保留继续使用
